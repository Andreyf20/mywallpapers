## Instructions

To resize new images to at least 4K resolution (3840x2160) while preserving their aspect ratio, you can use the provided `upscale.py` script.

1. Ensure you have Python 3 installed.
2. Install the required dependency:
   ```bash
   pip install pillow
   ```
3. Run the script:
   ```bash
   python upscale.py
   ```

The script will automatically detect images smaller than 4K, scale them using high-quality Lanczos resampling, and update the table in this `README.md` with their new resolution and status.
